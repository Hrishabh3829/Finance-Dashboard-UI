# Features
- Dashboard overview with summary cards, time-based trend, and category breakdown
- Transactions list with search, sorting, pagination, advanced filters, and export (CSV/JSON)
- Role-based UI (Viewer/Admin) with admin-only transaction actions
- Insights section for spending highlights and simple comparisons
- Drawer-based advanced filters and animated theme toggle
- Loading states and skeletons for polished UX

# Notes
This project is intentionally simple and focused on UI/UX clarity. Data is mocked on the client and persisted in localStorage so the dashboard feels real without a backend.
