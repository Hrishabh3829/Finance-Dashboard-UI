## 💳 Finance Dashboard (React + Vite + Tailwind)

A responsive finance dashboard that helps users track transactions, analyze spending patterns, and gain quick insights through a clean and intuitive interface.

---

## 📌 Project Overview

This project demonstrates how to build a modern frontend dashboard with:

* Data visualization
* Transaction management
* Role-based access control
* Advanced filtering and export functionality

The application uses client-side data (mocked) and persists it using localStorage to simulate a real-world experience without backend complexity.

---

## Tech Stack

* React.js
* Vite
* Tailwind CSS
* JavaScript (ES6+)
* LocalStorage

---

## Features

### Dashboard

* Summary cards for financial overview
* Time-based spending trends
* Category-wise expense breakdown

---

### Transactions

* Search, sorting, and pagination
* Advanced filters via a drawer interface
* Export data as CSV and JSON

---

### Role-Based Access

* Viewer: read-only access
* Admin: manage transactions

---

### Insights

* Highlights key spending patterns
* Simple comparisons for better understanding

---

### User Experience

* Responsive design across devices
* Skeleton loaders for loading states
* Light/Dark theme toggle
